# Healthcare Management System

## How to Run
1. Ensure all necessary files are in the `data/` directory.
2. Build the program:
   - Use the VS Code task "Build Healthcare System" or run `make` (if using Makefile).
3. Run the program:
   - Use the VS Code task "Run Healthcare System" or execute the binary.

## Features
- Add/Delete/Update/Search Doctors and Appointments.
- Query data using indexes.
